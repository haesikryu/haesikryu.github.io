# AI와 기술의 새로운 동향
최근의 기술 뉴스에서 가장 중요한 이야기를 몇 가지 정리해봤습니다. 

## 1. GLM 5.2 출시
**요약:** GLM-5.2는 실제 사용 가능한 1M 컨텍스트와 장기 과제 독립 수행을 지원하는 오픈소스 모델로, 복잡한 에이전트 애플리케이션 구축 기반을 제공합니다. 프런티어 모델 접근이 비기술적 이유로 갑자기 차단되는 상황에서 과학은 세계적이어야 하며 AGI로 가는 길은 높은wall을 Scaling합니다.
**중요성:** 이 모델의 출시와 함께, 더 복잡하고 강력한 AI 애플리케이션의 구축이 가능해질 것으로 예상되며, 이는 다양한 산업과 분야에 큰 영향을 미칠 것입니다.
**출처:** [https://news.hada.io/topic?id=30478](https://news.hada.io/topic?id=30478)

## 2. 이메일의 미래
**요약:** AI가 받은 편지함을 읽고 요약하며 작업까지 수행하는 환경에서는 발신자 검증이 이메일 신뢰의 핵심 조건이 됩니다. SPF, DKIM, DMARC는 서버 권한, 메시지 무결성, 실패 시 처리 방침을 결합해 이메일 인증의 기본 구조를 이룹니다.
**중요성:** 이메일의 будущ에는 보안과 인증이 더욱 중요해질 것으로 예상되며, 이러한 기술의 발전은 온라인 통신의 신뢰성을 높일 것입니다.
**출처:** [https://news.hada.io/topic?id=30477](https://news.hada.io/topic?id=30477)

## 3. repo-slopscore: 커밋 기록 분석으로 Git 저장소의 AI/LLM 기여 감지
**요약:** repo-slopscore는 Git 저장소의 커밋 기록을 분석해 AI/LLM 기여를 감지하는 도구로 소개됩니다. 서비스는 홈, 저장소 스캔, 소스 코드 링크를 제공하며 소스 코드는 codeberg.org/polyphony/repo-slopscore에 공개됩니다.
**중요성:** 이 도구는 Git 저장소에서 AI/LLM의 기여도를 감지하는 데 도움이 될 것입니다. 이는 AI의 영향력과 기여도를 더 잘 이해하는 데 중요한 역할을 할 것입니다.
**출처:** [https://news.hada.io/topic?id=30475](https://news.hada.io/topic?id=30475)
