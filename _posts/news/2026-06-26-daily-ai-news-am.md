## 2. 새로운 HTTP QUERY 메소드 — 더 효율적인 검색
**요약:** RFC 10008로 새롭게 정의된 HTTP QUERY 메소드는 기존 RESTful API에서 복잡한 검색을 처리할 때(GET과 POST의) 한계를 해결하기 위해 논의 끝에 표준화되었습니다. QUERY 메소드는 더 효율적인 검색을 ermöglicht.
**왜 중요한가:** QUERY 메소드는 개발자들이 더 효율적인 검색을 구현할 수 있도록 도와주어 사용자 경험을 개선할 수 있습니다.
